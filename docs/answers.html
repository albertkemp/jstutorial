Question 11:

Answer: plants.slice(1, 3); or console.log(plants.slice(1, 3));

Explanation: The .slice() method specifies the starting point and the ending point. the starting point is plants[1] which is tree, and the ending point is plants[2], which is bush. Since it cuts that portion out, in between is just the string 'tree'.
